# Agent Bootstrap

This repo is operated by a file-driven multi-agent workflow. Whatever tool,
model, or harness you are running under:

1. Read `CLAUDE.md` — the operating manual. It defines model routing, the
   ask-before-execute protocol, and where all state lives.
2. Read `.claude/state/index.md` — the live task map.
3. Work only through that protocol. Store everything you learn in the files
   it names, locally in this repo — never only in conversation memory.
